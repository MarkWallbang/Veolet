# Veolet

## Install & Import

import as:

```
import(
...
github.com/markwallbang/veolet
)
```


## Flow emulator

Command to start emulator:

```
$ flow emulator start -v
```

Command to deploy project on emulator:

```
flow project deploy --network=emulator
```

## Run tests

Tests can be run with the following command:

```
go test -v
```
